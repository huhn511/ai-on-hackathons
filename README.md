## 📖 About This Presentation

An interactive presentation showcasing how AI agents are transforming software development for developers and makers. Features a live Gemini CLI demonstration building Flappy Bird from scratch, explores the Model Context Protocol (MCP) as "USB-C for AI", and looks ahead to the future of conversational programming and physical AI integration.

### 🎯 Core Topics

- **📚 AI Evolution Timeline**: From ELIZA (1966) → ChatGPT (2022) → MCP (2024) → Agent Revolution (2025+)
- **🔌 MCP Protocol**: "USB-C for AI" - universal connectivity for AI agents
- **🛠️ CLI Agents**: Claude Code, Gemini CLI, GitHub Copilot CLI showcasing terminal-native AI
- **🎮 Live Demo**: Building Flappy Bird with Gemini CLI - 3 prompts to playable game
- **🤖 Conversational Programming**: Natural language as the new programming interface  
- **🌍 Physical AI**: AI agents meeting the physical world through robotics
- **🚀 Future Vision**: Natural language programming becoming mainstream by 2027+

## 🏃‍♂️ Quick Start

```bash
# Clone the repository  
git clone https://github.com/huhn511/ai-on-hackathons.git
cd ai-on-hackathons

# Install dependencies
npm install

# Start development server (includes video support)
npm run dev

# Open http://localhost:3030
# Navigate through slides, watch embedded demo videos
```
